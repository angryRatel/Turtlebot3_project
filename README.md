## 디지털 트윈 기반 서비스 로봇 시스템 구성
> ROS2 기반의 자율 주행 라인 트레이싱과 ArUco 마커 인식을 통한 로봇팔 픽앤플레이스 작업을 수행하는 서비스 로봇 시스템입니다.
> 
### detect_lane에 중점을 두고 aruco_detect로 Manipulation을 작동하는 것을 서브로 프로젝트를 진행함
